# TAI NW KERSTBINGO

Deze tool is een interactieve terminal-applicatie voor een kerstquiz of kerstbingo.
Via een CSV-bestand worden vragen één voor één getoond, aangevuld met een feestelijk kerstsfeerthema in de terminal.

Tijdens het doorlopen van de vragen:
- Verschijnt telkens slechts één vraag in beeld (oude inhoud verdwijnt).
- Wordt na ENTER het antwoord getoond.
- Kan optioneel een afbeelding fullscreen geopend worden (bijvoorbeeld een kerstplaatje).
- Wordt elke behandelde vraag automatisch afgevinkt in de CSV.
- Kan de gebruiker met een getal aangeven welke categorie vraag er weergegeven moet wornden.

De tool herhaalt dit totdat alle vragen zijn afgehandeld.
